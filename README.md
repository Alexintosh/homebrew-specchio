# Alexintosh Specchio Homebrew Tap

This tap contains the [Specchio](https://github.com/Alexintosh/specchio-cli) CLI formula.

## Installation

```bash
brew tap Alexintosh/specchio
brew install specchio
```

## Usage

```bash
specchio              # Interactive setup wizard
specchio check        # Run all checks
specchio doctor       # Show diagnostics
specchio verify       # Verify setup
```

See https://github.com/Alexintosh/specchio-cli for full documentation.
